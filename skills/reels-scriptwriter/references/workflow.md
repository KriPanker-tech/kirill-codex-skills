# Reels Scriptwriter Workflow

## When To Read This

Read this file when:

- building a content plan
- writing more than one script
- adapting the system to a niche
- improving an existing batch of ideas

## Fast Workflow

1. Define the business role of the video:
   - reach
   - trust
   - sell
   - move to funnel

2. Choose the awareness level:
   - unaware
   - problem-unaware
   - problem-aware but passive
   - solution-seeking
   - provider-choice

3. Pick the main metric:
   - 4-second hold
   - retention
   - follow conversion
   - comments
   - shares/saves

4. Write 3 hook options.

5. Choose the strongest structure:
   - list
   - liver-cake
   - story
   - instruction
   - one-step fix

6. Add one CTA only.

## Script Output Format

Use this shape when the user wants a full script:

- Goal:

## Content Plan Output Format

Use a compact table-like list:

- idea
- role
- Hunt level
- format
- hook angle
- CTA direction

## Rewrite Moves

If a draft is weak, usually improve in this order:

1. Cut abstraction
2. Sharpen first line
3. Reduce setup
4. Restore one main idea
5. Delay the answer
6. Make CTA more specific
