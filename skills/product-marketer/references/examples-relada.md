# Example: RELADA PAEI

Use `RELADA PAEI - разбор проекта и консалт-контур` as a reference for:

- segment split when one product tries to speak to several very different audiences
- building a matrix `segment -> job -> value -> offer -> CTA`
- separating channel questions from segment and value questions
- diagnostic sprint logic before a full-scale marketing build

Key lesson:

If segment and value are still blurred, selling a full marketing system too early creates noise and wasted spend.
