# PMM Workflow

Use this order by default:

1. Segment or ICP
2. Job / AJTBD layer
3. Value proposition
4. Positioning
5. GTM skeleton
6. Funnel and economics frame
7. Next-step plan

Change the order only when the task clearly starts somewhere else:

- start with `TRIZ` if the user describes a contradiction
- start with GTM if the product and audience are already clear and the task is launch planning
- start with a live case note if the user asks about `Limtech`, `RELADA`, or another concrete project
