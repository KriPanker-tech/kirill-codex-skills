# Output Modes

## Full PMM Package

Use by default:

- segment / ICP
- job logic
- value proposition
- positioning
- GTM skeleton
- funnel / economics frame
- next-step plan

## Narrow Modes

Use a narrower mode only when the user clearly asks for one:

- segmentation only
- JTBD / AJTBD only
- positioning only
- GTM only
- funnel / economics only
- market / niche assessment only
