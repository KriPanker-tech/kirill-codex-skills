# Framework Routing

## Use AJTBD when

- the core problem is segment selection
- the user needs to understand what job the customer is trying to get done
- the task involves interviews, path to purchase, triggers, barriers, or decision criteria

## Use TRIZ when

- the user faces a contradiction
- the system seems blocked by a tradeoff
- ordinary marketing tactics are not enough

## Use the GTM template when

- the task needs a structured launch or growth plan
- research, positioning, channels, and economics must be tied together
- the output should look like a working GTM frame rather than loose notes
